# Persepolis Circular OS — Next.js

Site institucional criado em Next.js + React + TypeScript.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Gerar versão para publicação

```bash
npm run build
```

A versão estática será gerada na pasta `out/`.

## Publicar na Vercel

1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste projeto.
3. Entre na Vercel.
4. Importe o repositório.
5. Clique em Deploy.

## Publicar no GitHub Pages

Como o projeto usa `output: "export"`, ele gera arquivos estáticos.

1. Rode `npm install`.
2. Rode `npm run build`.
3. Publique o conteúdo da pasta `out/` no GitHub Pages.
